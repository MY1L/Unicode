# 统一码区段汉化
苦于译名混乱，对统一码[Blocks-13.0.0.txt](https://www.unicode.org/Public/UCD/latest/ucd/Blocks.txt)文件内区段名称作临时汉化。为适应旧软件，汉化的文件编码不是UTF。尚未完全考据所以有部分未区分“某”文、“某”字母、“某”语文字。
